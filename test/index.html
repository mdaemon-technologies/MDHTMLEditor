Custom Toolbar Buttons

These buttons were added via editor.ui.registry.addButton() in the setup callback and included in the toolbar string.

setup: (ed) => { // Text-label button ed.ui.registry.addButton('wordcount', { text: '💬 Words', tooltip: 'Show word count', onAction: () => { /* ... */ }, }); // Icon (SVG data-URI) button ed.ui.registry.addButton('timestamp', { icon: 'data:image/svg+xml,...', tooltip: 'Insert timestamp', onAction: () => { /* ... */ }, }); // Toggle button with active state ed.ui.registry.addButton('markpen', { text: '🖍️', tooltip: 'Toggle mark pen', onSetup: (api) => { api.setActive(false); }, onAction: (api) => { api.setActive(!api.isActive()); }, }); } // toolbar must list the names: toolbar: '... wordcount timestamp markpen'
